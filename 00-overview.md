# AI Project Context Kit

คู่มือนี้ใช้สำหรับโปรเจกต์ที่มีหลายคน contribute และแต่ละคนใช้ AI coding tool คนละตัว เช่น Cursor, GitHub Copilot, Claude Code, Codex, Cline, Continue, Aider หรือ Llama/Ollama/local model

เป้าหมายคือทำให้ AI ทุกตัวทำงานตามมาตรฐานเดียวกัน ไม่สร้าง pattern คนละแบบ และไม่แก้ไฟล์เกิน scope
